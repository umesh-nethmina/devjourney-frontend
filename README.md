# 🌟 DevJourney Frontend

**DevJourney** is a gamified developer career simulator where users learn coding by completing real-world projects, progressing through career levels, and interacting with AI mentors.

---

## 🚀 Features

* 🎮 Gamified Dashboard (XP, Coins, Badges, Levels)
* 🧑‍💻 VS Code-like Code Editor (Monaco Editor)
* 🌳 Career Ladder & Skill Tree
* 📋 Task System (Frontend, Backend, Full-stack)
* 🤖 AI Mentor & AI Reviewer Integration
* 🧑‍🤝‍🧑 Multiplayer Hackathon Rooms
* 🎉 Seasonal Events & Rewards
* 📊 Analytics Dashboard

---

## 🛠️ Tech Stack

* **Framework:** Next.js (React)
* **Styling:** Tailwind CSS
* **State Management:** Zustand / Redux Toolkit
* **Animations:** Framer Motion
* **Code Editor:** Monaco Editor
* **Realtime:** Socket.IO / WebRTC

---

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/devjourney-frontend.git
cd devjourney-frontend
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment variables

Create `.env.local` file:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000
```

### 4. Run development server

```bash
npm run dev
```
